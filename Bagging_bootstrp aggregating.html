Label 설명 (영문 → 한글)

  • Original data → 원래 학습 데이터 (훈련 세트 전체)
  • Bootstrap sample #k → 복원추출로 만든 k번째 부트스트랩 표본
  • Tree #k vote → k번째 의사결정나무가 테스트 샘플에 대해 내린 예측(투표)
  • Ensemble vote → 여러 트리의 투표를 합친 최종 다수결 예측
  • Class A / Class B → 두 개의 클래스(파란색/주황색)를 의미
  • Test sample → 레이블을 모르는 새로운 데이터 포인트

애니메이션은 배깅(Bagging)이 어떻게 여러 부트스트랩 샘플에서 트리를 학습시키고, 각 트리의 예측을 모아서(Test sample에 대한 투표) 최종 예측을 만드는지 시각적으로 보여 줍니다. Play 버튼을 누르면
(1) 원본 데이터에서 복원추출로 여러 표본을 뽑고,
(2) 각 표본으로 트리를 학습해 투표를 만들며,
(3) 마지막에 Ensemble vote가 다수결 결과로 활성화되는 과정을 볼 수 있습니다.
